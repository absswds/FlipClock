# FlipClock Web

React, TypeScript, and Vite version of FlipClock.

## Commands

```bash
npm install
npm run dev
npm run test
npm run lint
npm run build
```

## Structure

```text
src/
|-- components/   # Screens and reusable flip display components
|-- hooks/        # Timer and settings state
|-- logic/        # Pure formatting, i18n, theme, and clock-state helpers
`-- styles/       # Main CSS
```

## Build Output

`npm run build` writes static files to `dist/`.

The root README contains the public project overview and release notes.
